# Changelog

## 1.2.0 - 2026-06-06

- Репозиторий переименован в `mcp-sentry`; раскатка через `npx github:` и ветку `dist` как в `mcp-atlas`.
- TypeScript компилируется в `build/`; скрипт сборки — `compile` (не `build`, npm/cli#4003).
- CI на push в `master`: тесты, публикация `dist`, git-тег `v{version}`.
- Добавлены `AGENTS.md`, `mcp-config.example.json`, `.env.example`, jest-тесты.

## 1.1.0 - 2026-04-20

- Добавлен инструмент `export_issue_events_to_file` для выгрузки всех событий issue с указанной UTC-даты в локальный `JSONL`-файл с возвратом пути к файлу.
- Добавлен инструмент `extract_issue_event_fields_to_file` для извлечения полей вроде `requestParams` из событий issue в локальный `JSONL`-файл с возвратом пути к файлу.
- Добавлена постраничная загрузка событий issue и параллельная догрузка полных данных событий для больших выгрузок.
- В записи экспорта добавлены производные поля `exceptionMessage` и `eventTitle`, чтобы различать события с одинаковым названием issue.

## 1.0.0 - 2025-07-30

- Добавлен MCP-сервер для работы с Sentry API по `stdio`.
- Добавлены инструменты для просмотра проектов организации и списка issue проекта.
- Добавлены инструменты для получения детальной информации об issue и его событиях.
- Добавлен инструмент для получения конкретного события issue с дополнительными данными.
- Добавлен инструмент для разрешения `short_id` issue в полные данные Sentry.
- Добавлен инструмент для извлечения данных контекста из всех событий issue.
